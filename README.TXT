PURPOSE OF PROJECT:
It is an object-oriented Java program that models a recipe system.
You have a RecipeBook, Recipe, Ingredients, unitOfMeasurement classes. 2 Enums
and 3 subclasses, snacks, desserts, drinks. You are able to store recipes and 
organize them. The user is able to add view ingredients and recipe details. 

HOW TO START THIS PROJECT:
To run the project outisde of bluej,
-user must have Java downloaded in their personal computer.
-locate the file
-right click and click on run in terminal
-write "javac RecipeBook.java" to compile
-write "java RecipeBook" to execute the program
-have fun with the program!
-*program can be ran if user has BlueJ installed.
